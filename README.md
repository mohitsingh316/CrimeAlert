
# CrimeAlert

CrimeAlert is a mobile application aimed at enhancing community safety and crime awareness. It provides real-time crime updates, emergency support features, community engagement tools, and prevention tips to empower users and foster a safer community environment.

## Features

- Real-time Crime Updates: Get immediate access to the latest crime incidents in your area.
- Emergency Support: Quickly access emergency contacts and signal nearby users in danger.
- Community Engagement: Join local community groups, report suspicious activities, and access prevention tips.
- Prevention Tips: Stay informed with proactive measures to reduce the risk of becoming victims of crime.

## Technologies Used

- Android Studio
- Java/Kotlin
- MVVM Architecture
- Android SDK
- Jetpack Components
- Retrofit
- Firebase
- Git
- SQLite
